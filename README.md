# 🌐 مشروع محاكاة أتمتة الشبكات

هذا المشروع يهدف إلى تطبيق مفاهيم أتمتة الشبكات (Network Automation) باستخدام Python من خلال إنشاء بيئة شبكات افتراضية تحاكي الأجهزة والإعدادات بدون الحاجة إلى أجهزة حقيقية أو برامج محاكاة خارجية.

يقوم المشروع بإدارة أجهزة متعددة، وحفظ إعداداتها، وإنشاء نسخ احتياطية تلقائية، واكتشاف التغييرات في الإعدادات، وإنشاء التنبيهات والتقارير، بالإضافة إلى دعم جدولة المهام بشكل دوري وتوفير لوحة تحكم ويب (Web Dashboard) لمتابعة حالة النظام.

## 📁 محتويات المشروع

| الملف                      | الوصف                     |
| -------------------------- | ------------------------- |
| `main.py`                  | الملف الرئيسي للمشروع     |
| `devices.json`             | بيانات الأجهزة الافتراضية |
| `device.py`                | تعريف جهاز الشبكة         |
| `simulator.py`             | إدارة ومحاكاة الأجهزة     |
| `device_loader.py`         | تحميل الأجهزة من JSON     |
| `backup_manager.py`        | إنشاء النسخ الاحتياطية    |
| `config_manager.py`        | حفظ وإدارة الإعدادات      |
| `diff_engine.py`           | مقارنة الإعدادات          |
| `report_generator.py`      | إنشاء التقارير            |
| `alert_manager.py`         | إنشاء التنبيهات           |
| `scheduler.py`             | جدولة المهام              |
| `logger.py`                | تسجيل العمليات            |
| `web/app.py`               | لوحة التحكم               |
| `web/templates/index.html` | واجهة Dashboard           |
| `web/static/style.css`     | تنسيقات Dashboard         |
| `data/configs/`            | ملفات الإعدادات           |
| `data/backups/`            | النسخ الاحتياطية          |
| `data/reports/`            | التقارير والتنبيهات       |
| `reports/`                 | ملفات السجل               |
