# 🌐 مشروع محاكاة أتمتة الشبكات

هذا المشروع يهدف إلى تطبيق مفاهيم أتمتة الشبكات (Network Automation) باستخدام Python من خلال إنشاء بيئة شبكات افتراضية تحاكي الأجهزة والإعدادات بدون الحاجة إلى أجهزة حقيقية أو برامج محاكاة خارجية.

يقوم المشروع بإدارة أجهزة متعددة، وحفظ إعداداتها، وإنشاء نسخ احتياطية تلقائية، وإنشاء تقارير عن الأجهزة الموجودة داخل النظام، بالإضافة إلى اكتشاف التغييرات بين الإعدادات وإطلاق تنبيهات عند حدوث أي تعديل.

---

## 🛠️ التقنيات المستخدمة

* **Python 3**
* **JSON** – تخزين بيانات الأجهزة
* **Object-Oriented Programming (OOP)** – بناء الأجهزة والمحاكاة
* **File Handling** – حفظ الإعدادات والتقارير
* **Logging** – تسجيل العمليات والأحداث
* **Difflib** – مقارنة التغييرات بين الإعدادات

---

## 📁 محتويات المشروع

| الملف                 | الوصف                              |
| --------------------- | ---------------------------------- |
| `main.py`             | الملف الرئيسي لتشغيل المشروع       |
| `devices.json`        | بيانات الأجهزة الافتراضية          |
| `device.py`           | تعريف جهاز الشبكة                  |
| `simulator.py`        | إدارة ومحاكاة الأجهزة              |
| `device_loader.py`    | تحميل الأجهزة من ملف JSON          |
| `backup_manager.py`   | إنشاء النسخ الاحتياطية             |
| `config_manager.py`   | حفظ وإدارة الإعدادات               |
| `diff_engine.py`      | مقارنة الإعدادات واكتشاف التغييرات |
| `report_generator.py` | إنشاء تقارير التغييرات             |
| `logger.py`           | تسجيل العمليات والأحداث            |
| `data/configs/`       | الإعدادات المحفوظة                 |
| `data/backups/`       | النسخ الاحتياطية للأجهزة           |
| `data/reports/`       | تقارير التغييرات                   |
| `reports/`            | ملفات السجل                        |
